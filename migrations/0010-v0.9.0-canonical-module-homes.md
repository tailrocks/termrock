# Migration 0010: canonical module homes

TermRock now exposes each concept through one public path. The `geometry`
junk-drawer module, crate-root utility re-exports, `theme` alias, and duplicate
root pointer API are removed.

## Pointer API

Delete imports of the removed root `termrock::PointerShape`. Use
`termrock::osc::PointerShape`, whose canonical variant set is now `Default`,
`Pointer`, `Text`, `Crosshair`, `EwResize`, `NsResize`, and `Grabbing`.

Replace:

```rust
use termrock::{PointerShape, osc22_pointer_shape};

let sequence = osc22_pointer_shape(PointerShape::EwResize);
```

with:

```rust
use termrock::osc::{PointerShape, encode_pointer};

let sequence = String::from_utf8(encode_pointer(PointerShape::EwResize))?;
```

The removed `clickable_pointer_shape(clickable)` helper becomes the direct
mapping appropriate to the caller:

```rust
let shape = if clickable {
    PointerShape::Pointer
} else {
    PointerShape::Default
};
```

## Import example

Before:

```rust
use termrock::{HintSpan, TailScroll, centered_rect, display_cols};
use termrock::theme::Role;
```

After:

```rust
use termrock::{
    layout::centered_rect,
    scroll::TailScroll,
    style::Role,
    text::display_cols,
    widgets::HintSpan,
};
```

No rendering, width calculation, sanitization, scrolling, tab geometry, or
hint behavior changed. Only ownership and public paths changed.
