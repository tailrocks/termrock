# 0028 — VirtualGrid widget (additive)

## Removed surface

None. Additive public API only.

## Canonical replacement / addition

New public types under `termrock::widgets`:

- `VirtualGrid`, `VirtualGridState`, `VirtualGridOutcome`
- `GridColumn`, `GridColumnWidth`, `GridRow`, `GridCell`
- `GridCellRegion`, `GridHeaderRegion`

## Consumer edits

Consumers that need a two-axis virtualized data grid should compose
`VirtualGrid` with a windowed provider of borrowed cells. Do not build a
local generic grid substitute.

## Ownership

TermRock owns viewport, selection, hit regions, column width resolution, and
Unicode-safe cell paint. Callers own fetching, editing, sort/filter, and
domain models.

## Validation

```bash
cargo test -p termrock virtual_grid
cargo test --workspace
cargo run -p termrock-lookbook -- check --dir docs/public/component-previews
```
