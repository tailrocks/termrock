# 0015: Independent terminal session options

`crossterm::SessionOptions` now controls alternate-screen ownership, cursor
visibility, and line wrapping independently. `hide_cursor` and
`disable_line_wrap` both default to `true`, preserving the default session.

Adding public fields breaks exhaustive struct literals. Prefer a default-based
literal so future session capabilities inherit their defaults:

```rust
// before
let options = SessionOptions {
    alternate_screen: false,
    mouse_capture: false,
    bracketed_paste: false,
    raw_mode: false,
};

// after: inline UI with a hidden cursor, normal line wrapping
let options = SessionOptions {
    alternate_screen: false,
    hide_cursor: true,
    disable_line_wrap: false,
    mouse_capture: false,
    bracketed_paste: false,
    raw_mode: false,
};

// recommended when only selected behavior differs
let options = SessionOptions {
    alternate_screen: false,
    disable_line_wrap: false,
    ..SessionOptions::default()
};
```

Terminal cleanup still tracks every acquired mode separately and restores it
in reverse acquisition order.
